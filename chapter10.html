생성자 함수와 this 노트 정리

작성일: 2025년 5월 20일 오후 10:08 KST

생성자 함수를 사용해 객체 정의하기

- 생성자 함수는 일반적인 함수와 같은 형식으로 정의한다.
- 함수 이름의 첫 글자는 대문자로 사용한다.
- 함수 내부에서 `this`를 사용한다. 

function 함수명(매개변수) {
    this.키1 = 값1;
    this.키2 = 값2;
    this.메서드1 = function() { ... };
    this.메서드2 = function() { ... };
}

예시

- 객체를 만들 때 사용하는 `new` 키워드 예시.
- 사용자가 정의한 객체와 빌트인 객체(예: `Date`)를 활용. 

const 함수명(매개변수) = function () {
    ...
}

생성자 함수를 사용해 객체 만들기

- 객체를 만들 때 `new` 키워드를 사용해 인스턴스를 생성.
- 예: `new 함수명()`으로 객체를 생성. 

let book4 = {
    title: "Javascript",
    pages: 500,
    author: "고정희",
    done: false,
    finish: function() {
        this.done === false ? console.log("읽는 중") : console.log("완독");
    }
}

book4.finish() // "읽는 중"

객체 복사하기

- 객체의 속성 값을 다른 객체에 복사.
- 예: `number2 = number1`로 값 복사 후 수정. 

let number1 = 100;
let number2 = number1; // 100
number2 = 200; // number1은 100, number2는 200

- 객체의 참조 복사 예시 (색상 변경).
- `bag1 = bag2`로 참조가 동일하게 동작. 

let bag1 = { color: "blue" };
let bag2 = bag1; // 참조 복사
bag2.color = "red"; // bag1.color도 "red"로 변경

메서드와 this

- `this`: 메서드에서 객체 자체를 참조.
- 예: `student.average()`에서 `this`로 속성 접근. 

let student = {
    name: "DoReMi",
    score: {
        history: 85,
        science: 94,
        average: function() {
            return (this.history + this.science) / 2;
        }
    }
}

student.score.history // 85
student.score.average() // 89.5

객체 복사하기 (참조와 값)

- 참조: `let student = { ... }`로 객체 생성 후 수정.
- 값 복사: `let copy = { ...student }`로 깊은 복사 가능.
- DOM 객체: 문서 객체 모델(DOM) 활용 예시. 

객체 명 {
    키1: 값1,
    키2: 값2,
    ...
}

요약

- 생성자 함수는 `new` 키워드로 객체를 생성하며, `this`로 속성/메서드 정의.
- 객체 복사는 값 복사와 참조 복사가 가능하며, 주의가 필요.
- `this`는 메서드 내에서 현재 객체를 가리킴.
- DOM 활용으로 동적 웹 페이지 제어 가능.
