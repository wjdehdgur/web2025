Chapter 12: HTML5 및 Canvas API

HTML5와 PC와 모바일

  • 2012년 국내 최초로 HTML5 게임 개발을 시작하여 웹과 웹, PC와 모바일 어느 환경에서도 구동 가능.
  • 삼성전자, 페이스북 같은 글로벌 파트너와 함께 진행.
  • 게임은 모든 구성을 기반으로 게임을 만들며 환경에 맞춰 조정.

캔버스 사용

  • HTML에 포함된 Canvas API를 사용하면 웹 문서 상에 직접 그려볼 수 있다.
    • 예: <canvas> 태그를 사용하여 2D 그래픽을 그린다.
  • Canvas API에는 도형을 그리거나 색칠하는 함수를 비롯해 선의 굵기나 색상 속성처럼 필요한 기능을 제공.
  • UI나 차트 같은 그래픽 인터페이스를 그려낼 수 있다.

캔버스 색상과 선 설정

  • 색상 설정
    • fillStyle = "색상": 내부를 채우는 색상 설정.
    • strokeStyle = "색상": 선의 색상 설정.
  • 예시
    • ctx.fillStyle = "yellow"; 내부를 노란색으로 채움.
    • ctx.strokeStyle = "red"; 선을 빨간색으로 설정.

도형 그리기

  • 기본 도형 그리기
    • beginPath(): 새로운 경로 시작.
    • closePath(): 현재 경로를 닫음.
    • moveTo(x, y): 펜을 (x, y) 위치로 이동.
    • lineTo(x, y): (x, y) 위치로 선을 그림.
    • arc(x, y, radius, startAngle, endAngle, counterclockwise): 원 또는 원호를 그림.
    • fill(): 내부를 채움.
    • stroke(): 선을 그림.
  • 예시
    • ctx.arc(200, 150, 100, 0, Math.PI * 2, true); 원을 그리며 내부를 노란색으로 채움.
    • ctx.arc(200, 200, 50, 0, Math.PI / 180 * 60, false); 원호를 그리며 선을 빨간색으로 설정.

캔버스 초기화 및 그리기

  • 초기화
    • clearRect(x, y, width, height): 지정된 사각형 영역을 지움.
  • 그리기 예시
    • ctx.beginPath(); ctx.moveTo(120, 200); ctx.arc(120, 200, 50, Math.PI / 180 * 90, (Math.PI / 180) * 270, false); ctx.stroke(); 파란색 원호 그림.
    • ctx.beginPath(); ctx.arc(200, 200, 50, 0, Math.PI / 180 * 60, false); ctx.stroke(); 빨간색 원호 그림.
