Chapter 2

자바스크립트 역할

  1. 동적 처리
    • 이벤트에 따라 정보 변화
  2. 상호작용 처리
    • 이용자의 동작에 따른 처리
  3. 접근성 개선
    • 음성, 터치, 화면크기, 다양한 플랫폼
  4. 사용자 경험 개선 (UX)
    • 이용자에게 편리한 인터페이스 (UI)
  5. 애플리케이션 개발
    • 모바일/데스크탑 애플리케이션, 서버 애플리케이션

자바스크립트 작성 방법

  1. Inline JavaScript
    • <button onclick="add()">ADD</button>
  2. Internal JavaScript
    • <script>코드</script>
  3. External JavaScript
    • <script src="app.js"></script>
  4. 필요할 때 코드 생성 추가
    • 문서내의 자바스크립트 코드가, 현재 HTML 문서내에 자바스크립트 코드 추가 (코드가 코드를 추가)
      • 자바스크립트 코드로 컨텐츠, 태그, CSS, 코드 추가 가능
