# CSS Layout: `position` 속성 정리 CSS에서 `position` 속성은 요소의 위치를 제어하는 데 사용됩니다. 아래는 각 속성 값에 대한 설명입니다. ## 1. `position: static` - **기본값**: 요소가 순서에 따라 그대로 배치됩니다. - `top`, `left`, `right`, `bottom` 속성이 적용되지 않습니다. ## 2. `position: relative` - **상대 위치**: 요소가 원래 위치에서 상대적으로 이동합니다. - `top`, `left`, `right`, `bottom`으로 이동 거리를 지정합니다. - 원래 위치를 기준으로 이동하며, 다른 요소의 배치에는 영향을 주지 않습니다. ## 3. `position: fixed` - **고정 위치**: 스크롤이 되어도 화면의 고정된 위치에 유지됩니다. - 브라우저 화면(Viewport)을 기준으로 위치가 결정됩니다. - `top`, `left`, `right`, `bottom`으로 위치를 지정합니다. ## 4. `position: absolute` - **절대 위치**: 부모 엘리먼트를 기준으로 절대적인 위치에 배치됩니다. - 기준점: 가장 가까운 `relative`, `absolute`, `fixed`인 부모 요소. - 부모가 없으면 `body`를 기준으로 위치가 결정됩니다. ## 5. `position: sticky` - **고정 + 스크롤**: 화면에 보일 때는 `relative`처럼, 스크롤이 되어 안 보이게 되면 `fixed`처럼 동작합니다. - 기준점: `relative`처럼 작동. - 스크롤 영역을 벗어나면 `fixed`처럼 작동. - 예: 스크롤 내비게이션 메뉴나 특정 팝업 광고. --- ### 참고 - `position` 속성은 요소의 배치를 정밀하게 제어하는 데 유용합니다. - 각 속성의 동작을 이해하면 복잡한 레이아웃도 쉽게 구현할 수 있습니다.
