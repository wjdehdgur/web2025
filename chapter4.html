Chapter 2: 자바스크립트 개요

자바스크립트의 주요 역할

  • 웹 페이지의 동적 기능 구현
    • 사용자가 페이지를 새로고침하지 않아도 콘텐츠가 실시간으로 변경됩니다.
    • 예: 버튼 클릭 시 새로운 메시지가 표시되거나, 입력값에 따라 결과가 바로 업데이트됨.
  • 사용자와의 상호작용 지원
    • 마우스 클릭, 키보드 입력, 스크롤 등 다양한 사용자 동작에 반응합니다.
    • 예: 폼 제출 버튼 클릭 시 유효성 검사 후 알림창 표시.
  • 다양한 환경에서의 접근성 향상
    • 모바일, 태블릿, 데스크톱 등 다양한 디바이스에서 일관된 경험 제공.
    • 음성 인식, 터치스크린, 고대비 모드 등 접근성을 고려한 기능 지원.
  • 사용자 경험(UX) 최적화
    • 직관적이고 편리한 사용자 인터페이스(UI)를 제공합니다.
    • 예: 부드러운 애니메이션 효과, 즉각적인 피드백 제공.
  • 다양한 애플리케이션 제작
    • 웹 애플리케이션뿐만 아니라 모바일 앱, 데스크톱 앱 개발 가능.
    • Node.js를 활용해 서버 사이드 애플리케이션도 개발할 수 있음.

자바스크립트 작성 방법

  1. 인라인 자바스크립트
    • HTML 태그 안에 직접 자바스크립트 코드를 작성하는 방식.
    • 예: <button onclick="alert('클릭!')">클릭</button>
    • 간단한 기능에 적합하지만, 코드가 많아지면 관리하기 어려움.
  2. 내부 자바스크립트
    • HTML 파일 안에 <script> 태그를 사용해 코드를 작성.
    • 예: <script>function sayHello() { alert('안녕!'); }</script>
    • 작은 프로젝트에서 유용하며, HTML과 함께 관리 가능.
  3. 외부 자바스크립트
    • 별도의 .js 파일을 만들어 연결하는 방식.
    • 예: <script src="myScript.js"></script>
    • 코드 재사용성과 유지보수에 유리하며, 대규모 프로젝트에 적합.
  4. 동적으로 코드 생성 및 삽입
    • 자바스크립트를 사용해 실행 중에 새로운 코드를 추가.
    • 예: document.createElement()로 새로운 HTML 요소를 만들고 삽입.
    • 콘텐츠, 스타일(CSS), 추가 자바스크립트 코드 등을 동적으로 생성 가능.
    • 복잡한 웹 애플리케이션에서 자주 사용됨.
