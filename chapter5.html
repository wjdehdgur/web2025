Chapter 3: DOM(Document Object Model) 개요

DOM이란?

  • DOM(Document Object Model)은 HTML 문서를 객체로 표현한 구조입니다.
    • HTML 요소, 속성, 텍스트 등을 트리 구조로 나타냅니다.
    • 자바스크립트를 통해 동적으로 웹 페이지의 내용을 변경하거나 조작할 수 있습니다.
  • DOM은 브라우저가 HTML을 해석한 결과물입니다.
    • HTML 태그는 DOM에서 노드(Node)로 표현됩니다.
    • 예:
      태그는 요소 노드(Element Node)로 변환됩니다.

DOM의 주요 역할

  • 웹 페이지의 구조 접근 및 조작
    • 자바스크립트를 사용해 HTML 요소를 추가, 삭제, 수정 가능.
    • 예: document.getElementById('myId')로 특정 요소 선택.
  • 이벤트 처리
    • 사용자 동작(클릭, 마우스 오버, 키보드 입력 등)을 감지하고 반응.
    • 예: element.addEventListener('click', function() {...})로 클릭 이벤트 등록.
  • 스타일 동적 변경
    • CSS 스타일을 자바스크립트로 변경 가능.
    • 예: element.style.backgroundColor = 'blue'로 배경색 변경.
  • 콘텐츠 실시간 업데이트
    • 페이지 새로고침 없이 텍스트, 이미지 등 콘텐츠를 갱신.
    • 예: element.innerHTML = '새로운 텍스트'로 내용 변경.
  • 폼 데이터 처리
    • 사용자 입력값을 실시간으로 검증하거나 처리.
    • 예: inputElement.value로 입력값 가져오기.

DOM 조작 방법

  1. 요소 선택
    • 특정 HTML 요소를 선택해 조작.
    • 예: document.querySelector('.class')로 클래스 선택.
    • 다른 방법: getElementsByTagName(), getElementsByClassName().
  2. 요소 생성 및 추가
    • 새로운 요소를 만들어 문서에 삽입.
    • 예: document.createElement('p')로 새로운 단락 생성.
    • 삽입: parent.appendChild(newElement)로 추가.
  3. 요소 속성 변경
    • HTML 속성을 동적으로 수정.
    • 예: element.setAttribute('id', 'newId')로 ID 변경.
    • 속성 읽기: element.getAttribute('id').
  4. 노드 삭제
    • 필요 없는 요소를 제거.
    • 예: parent.removeChild(child)로 자식 노드 삭제.
    • 또는 element.remove()로 직접 삭제.
  5. 이벤트 연결
    • 사용자 동작에 반응하도록 이벤트 리스너 추가.
    • 예: button.addEventListener('click', () => alert('클릭!')).
    • 이벤트 제거: removeEventListener() 사용.
