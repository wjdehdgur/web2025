생성자 함수와 객체 노트 정리

생성자 함수를 사용해 객체 정의하기

  • 생성자 함수는 일반적인 함수와 같은 형식.
    • 함수 이름의 첫 글자는 대문자로 사용한다.
    • 함수 내부에서 this를 사용한다.
  • 형식 예시
    • function 함수명(매개변수) { this.키1 = 값1; this.키2 = 값2; this.메서드1 = function() { ... }; this.메서드2 = function() { ... }; }

생성자 함수를 사용해 객체 만들기

  • 객체마다 반복되는 프로퍼티와 메서드가 있다면 객체 특성을 미리 정의해 놓고 필요한 만큼 생성.
    • 특성을 사용해서 객체를 만들 수 있다.
  • 특정 객체를 만들 때 사용하는 함수를 생성자 함수라고 한다.
  • 생성자 함수를 사용해서 정의된 객체를 ‘인스턴스’ 또는 ‘인스턴스 객체’라고 부른다.
  • 예시
    • const 함수명(매개변수) = function () { ... }

객체 복사하기

  • 원시 유형 자료 복사
    • 값 복사: 값을 복사한다.
    • 복사한 자료의 값 변경 → 원래 자료의 값은 그대로.
    • 예시: let number1 = 100; let number2 = number1; number2 = 200; → number1은 100, number2는 200.
  • 객체 복사
    • 주소 복사: 주소를 복사한다.
    • 복사한 자료의 값 변경 → 원래 자료의 값도 바뀐다.
    • 예시: let bag1 = { color: "blue" }; let bag2 = bag1; bag2.color = "red"; → bag1.color도 "red"로 변경.

메서드와 this

  • this: 메서드에서 객체 안에 있는 프로퍼티값을 사용할 때, 현재 객체를 가리키는 예약어.
    • 예시: let book4 = { title: "Javascript", pages: 500, author: "고정희", done: false, finish: function() { this.done === false ? console.log("읽는 중") : console.log("완독"); } }; book4.finish(); → "읽는 중"

객체 참조하기

  • 객체 안에 또다른 객체를 넣을 수 있다.
    • 예시: let student = { name: "DoReMi", score: { history: 85, science: 94, average: function() { return (this.history + this.science) / 2; } } }; student.score.history; → 85, student.score.average(); → 89.5

사용자 정의 객체 만들기

  • 객체는 여러 개의 프로퍼티로 구성되어 있는데, 프로퍼티는 키: 값 형태를 가지고 있다.
  • 객체를 만들 때는 객체 이름 다음에 중괄호()를 사용하고, 중괄호 사이에 키: 값 형태로 필요한 프로퍼티를 나열한다.
  • 각 키는 문자열이거나 숫자, 심벌만 사용할 수 있고, 각 프로퍼티는 쉼표()로 구분한다.
  • 형식 예시
    • 객체 명 { 키1: 값1, 키2: 값2, ... }

객체 관리

  • 프로그램에서 객체 데이터를 저장하고 처리하는 기본 단위.
  • 자바스크립트에서 객체는 관련된 정보와 동작을 함께 모아 놓은 것.
  • 문서 객체 모델(DOM): 웹 문서 자체도 객체이고 웹 문서에 포함된 이미지와 링크, 텍스트 필드 등은 모두 두 이미지 객체, 링크 객체, 폼 객체처럼 각기 별도의 객체.
  • 브라우저 객체 모델: 웹 브라우저에서 사용할 정보 객체로 지정되어 있다.
  • 사용자 정의 객체: 필요한 때마다 사용자가 만들어 사용하는 객체.
